### 8.3 –£–≤–µ–ª–∏—á–µ–Ω–∏–µ —á–∏—Å–µ–ª –≤ —Å—Ç—Ä–æ–∫–µ (x10)
–ó–∞–º–µ–Ω–∏—Ç—å –≤—Å–µ —á–∏—Å–ª–∞ –≤ —Ç–µ–∫—Å—Ç–µ –Ω–∞ —á–∏—Å–ª–æ * 10. –ü–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è —Ü–µ–ª—ã—Ö –∏ –¥–µ—Å—è—Ç–∏—á–Ω—ã—Ö.
```py
import re

text = "I have 5 apples and 10 oranges, price is 0.5 each. Card number is ....3672."

def mul10(m: re.Match) -> str:
    s = m.group(0)
    if "." in s:
        return str(float(s) * 10)
    return str(int(s) * 10)

result = re.sub(r"\d+(?:\.\d+)?", mul10, text)
print(result)
```
