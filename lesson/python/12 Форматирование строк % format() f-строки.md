## 2) C-style —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ (–æ–ø–µ—Ä–∞—Ç–æ—Ä `%`)
### 2.1 –°–∏–Ω—Ç–∞–∫—Å–∏—Å
```py
"—Ñ–æ—Ä–º–∞—Ç–∏—Ä—É—é—â–∞—è —Å—Ç—Ä–æ–∫–∞" % (–∑–Ω–∞—á–µ–Ω–∏—è)
```

–§–æ—Ä–º–∞—Ç–∏—Ä—É—é—â–∞—è —Å—Ç—Ä–æ–∫–∞ —Å–æ–¥–µ—Ä–∂–∏—Ç **—Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã**, –Ω–∞—á–∏–Ω–∞—é—â–∏–µ—Å—è —Å `%`.

### 2.2 –ß–∞—Å—Ç—ã–µ —Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã
- `%s` ‚Äî —Å—Ç—Ä–æ–∫–∞ (–Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ: –ø–æ—á—Ç–∏ –ª—é–±–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –ø—Ä–µ–≤—Ä–∞—Ç–∏—Ç—Å—è –≤ —Å—Ç—Ä–æ–∫—É)
- `%d` ‚Äî —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ (`int`)
- `%f` ‚Äî —á–∏—Å–ª–æ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π (`float`)
- `%.2f` ‚Äî float —Å 2 –∑–Ω–∞–∫–∞–º–∏ –ø–æ—Å–ª–µ –∑–∞–ø—è—Ç–æ–π

### 2.3 –ü—Ä–∏–º–µ—Ä—ã
```py

age = 30
text = "My name is %s and I am %d years old." % (name, age)
print(text)

pi = 3.14159
text = "The value of pi is approximately %.2f." % pi
print(text)
```

### 2.4 –í–∞–∂–Ω–æ: —Ç–∏–ø—ã –¥–æ–ª–∂–Ω—ã –ø–æ–¥—Ö–æ–¥–∏—Ç—å
`%d` –∂–¥—ë—Ç —á–∏—Å–ª–æ, –Ω–µ —Å—Ç—Ä–æ–∫—É. –ï—Å–ª–∏ –ø–µ—Ä–µ–ø—É—Ç–∞—Ç—å ‚Äî –±—É–¥–µ—Ç –æ—à–∏–±–∫–∞.

```py

# –û—à–∏–±–∫–∞: %d –æ–∂–∏–¥–∞–µ—Ç —á–∏—Å–ª–æ, –Ω–æ –ø–æ–ª—É—á–∏—Ç —Å—Ç—Ä–æ–∫—É name
```

## 5) –§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —á–∏—Å–µ–ª (—Ç–æ—á–Ω–æ—Å—Ç—å, —Ç—ã—Å—è—á–∏, –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ)
–§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —á–∏—Å–µ–ª –æ–¥–∏–Ω–∞–∫–æ–≤–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –≤:
- `format()` ‚Üí `"{:.2f}".format(pi)`
- f-—Å—Ç—Ä–æ–∫–∞—Ö ‚Üí `f"{pi:.2f}"`

### 5.1 –ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –∑–Ω–∞–∫–æ–≤ –ø–æ—Å–ª–µ –∑–∞–ø—è—Ç–æ–π (`.nf`)
```py
pi = 3.14159
print(f"Pi: {pi:.2f}")                 # 3.14
print("Pi: {:.2f}".format(pi))         # 3.14
print("Pi: {num:.2f}".format(num=pi))  # 3.14
```

### 5.2 –†–∞–∑–¥–µ–ª–∏—Ç–µ–ª–∏ —Ç—ã—Å—è—á (`,`)
```py

```

## 6) –í—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ –∏ —à–∏—Ä–∏–Ω–∞ –ø–æ–ª—è (–≤ —Ñ–æ—Ä–º–∞—Ç-—Å–ø–µ–∫–µ)
–§–æ—Ä–º–∞—Ç-—Å–ø–µ–∫–∞ –≤ `{...:spec}`:
- `<` ‚Äî –≤–ª–µ–≤–æ
- `>` ‚Äî –≤–ø—Ä–∞–≤–æ
- `^` ‚Äî –ø–æ —Ü–µ–Ω—Ç—Ä—É
- —á–∏—Å–ª–æ = **—à–∏—Ä–∏–Ω–∞ –ø–æ–ª—è**

### 6.1 –ü—Ä–∏–º–µ—Ä—ã –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏—è

–¢–æ –∂–µ —á–µ—Ä–µ–∑ `format()`:
```py

print("start_{:<10}_end".format("text"))
print("start_{:^10}_end".format("text"))
```

### 6.2 –ú–∏–Ω–∏–º–∞–ª—å–Ω–∞—è —à–∏—Ä–∏–Ω–∞ –±–µ–∑ —è–≤–Ω–æ–≥–æ –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏—è
- –¥–ª—è —á–∏—Å–µ–ª –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é —á–∞—â–µ ‚Äú–≤–ø—Ä–∞–≤–æ‚Äù
- –¥–ª—è —Å—Ç—Ä–æ–∫ —á–∞—â–µ ‚Äú–≤–ª–µ–≤–æ‚Äù


### 6.3 –ó–∞–ø–æ–ª–Ω–µ–Ω–∏–µ (padding) –¥—Ä—É–≥–∏–º —Å–∏–º–≤–æ–ª–æ–º

#### 1. –ü—Ä–æ–¥–≤–∏–Ω—É—Ç–æ–µ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Å f-—Å—Ç—Ä–æ–∫–∞–º–∏
```python

# –í—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ —Ç–µ–∫—Å—Ç–∞

# –ü—Ä–æ—Ü–µ–Ω—Ç–Ω–æ–µ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ
```

```python
def progress_bar(percent, width=50):
    filled = int(width * percent / 100)
    bar = '‚ñà' * filled + '‚ñë' * (width - filled)
    return f"|{bar}| {percent:.1f}%"
```
