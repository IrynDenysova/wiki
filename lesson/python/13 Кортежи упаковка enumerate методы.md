–ï—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å—Ç–æ–ª—å–∫–æ –∂–µ, —Å–∫–æ–ª—å–∫–æ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö:

## 13) –î–æ–º–∞—à–Ω–µ–µ –∑–∞–¥–∞–Ω–∏–µ (—Ä–µ—à–µ–Ω–∏—è)

#### 4. –ò–º–µ–Ω–æ–≤–∞–Ω–Ω—ã–µ –∫–æ—Ä—Ç–µ–∂–∏ (namedtuple)
```python

# –ù–µ–∏–∑–º–µ–Ω—è–µ–º–æ—Å—Ç—å —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç—Å—è

# –ó–∞–º–µ–Ω–∞ –∑–Ω–∞—á–µ–Ω–∏–π (—Å–æ–∑–¥–∞–µ—Ç –Ω–æ–≤—ã–π –∫–æ—Ä—Ç–µ–∂)

# –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏–π –ø—Ä–∏–º–µ—Ä

    Person('–ê–ª–∏—Å–∞', 25, '–ú–æ—Å–∫–≤–∞'),

    Person('–í–∏–∫—Ç–æ—Ä', 28, '–ö–∞–∑–∞–Ω—å')
```
