## 9) –ö–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–ø–∏—Å–∫–æ–≤: —Å—Å—ã–ª–∫–∞ vs shallow copy vs deep copy
### 9.1 –°–∞–º–∞—è —á–∞—Å—Ç–∞—è –æ—à–∏–±–∫–∞: ¬´–∫–æ–ø–∏—è¬ª —á–µ—Ä–µ–∑ `=`

### 9.2 –ü–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ (shallow copy)
–°–æ–∑–¥–∞—ë—Ç –Ω–æ–≤—ã–π —Å–ø–∏—Å–æ–∫, –Ω–æ –≤–ª–æ–∂–µ–Ω–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∫–æ–ø–∏—Ä—É—é—Ç—Å—è –∫–∞–∫ **—Å—Å—ã–ª–∫–∏**.

–°–ø–æ—Å–æ–±—ã:
- `lst.copy()`
- `lst[:]`
- `list(lst)`

```py
original = [1, 2, 3]
copied = original.copy()
copied[0] = 99
print(original)  # [1, 2, 3]
print(copied)    # [99, 2, 3]
```

–ù–æ –Ω–∞ –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö —Å–ø–∏—Å–∫–∞—Ö shallow copy ¬´–Ω–µ —Å–ø–∞—Å–∞–µ—Ç¬ª:
```py
original = [[1, 2], [3, 4]]
shallow = original.copy()
shallow[0][0] = 99
print(original)  # [[99, 2], [3, 4]]  <- –ø–æ–º–µ–Ω—è–ª—Å—è —Ç–æ–∂–µ!
```

### 9.3 –ì–ª—É–±–æ–∫–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ (deep copy)
–°–æ–∑–¥–∞—ë—Ç –Ω–µ–∑–∞–≤–∏—Å–∏–º—É—é –∫–æ–ø–∏—é **–≤—Å–µ—Ö —É—Ä–æ–≤–Ω–µ–π –≤–ª–æ–∂–µ–Ω–Ω–æ—Å—Ç–∏**.

```py
import copy

original = [[1, 2], [3, 4]]
deep = copy.deepcopy(original)
deep[0][0] = 99

print(original)  # [[1, 2], [3, 4]]
print(deep)      # [[99, 2], [3, 4]]
```

‚úÖ –ö–æ–≥–¥–∞ —á—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:
- –æ–±—ã—á–Ω—ã–π —Å–ø–∏—Å–æ–∫ –±–µ–∑ –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö –∏–∑–º–µ–Ω—è–µ–º—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤ ‚Üí –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ shallow copy

#### 2. –ì–ª—É–±–∏–Ω–∞ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏—è –¥–ª—è —Å–ª–æ–∂–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä

#### –ü—Ä–∏–º–µ—Ä 3: –ö–µ—à–∏—Ä–æ–≤–∞–Ω–∏–µ —Å –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ–º

```python
import copy

class DataCache:
    def __init__(self):
        self._cache = {}
    
    def store(self, key, data):
        """–°–æ—Ö—Ä–∞–Ω—è–µ—Ç –≥–ª—É–±–æ–∫—É—é –∫–æ–ø–∏—é –¥–∞–Ω–Ω—ã—Ö"""
        self._cache[key] = copy.deepcopy(data)
    
    def retrieve(self, key):
        """–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –≥–ª—É–±–æ–∫—É—é –∫–æ–ø–∏—é –¥–∞–Ω–Ω—ã—Ö"""
        if key in self._cache:
            return copy.deepcopy(self._cache[key])
        return None

# –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ

data = [[1, 2], [3, 4]]
cache.store('matrix', data)

# –ò–∑–º–µ–Ω—è–µ–º –æ—Ä–∏–≥–∏–Ω–∞–ª
data[0][0] = 999

# –ö–µ—à –Ω–µ –∑–∞—Ç—Ä–æ–Ω—É—Ç
cached = cache.retrieve('matrix')
```

```python
def track_changes(original_list):
    """–û—Ç—Å–ª–µ–∂–∏–≤–∞–µ—Ç –∏–∑–º–µ–Ω–µ–Ω–∏—è —Å–ø–∏—Å–∫–∞"""
    snapshot = original_list.copy()
    
    def get_changes():
        added = [x for x in original_list if x not in snapshot]
        removed = [x for x in snapshot if x not in original_list]
        return {'added': added, 'removed': removed}
    
    return get_changes

# –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ

my_list.append(7)
```
