## 6) –ú–µ—Ç–æ–¥—ã –¥–æ–±–∞–≤–ª–µ–Ω–∏—è/—É–¥–∞–ª–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤

## 8) –û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ (union / intersection / difference / symmetric difference)

### 8.1 –í–∏–¥—ã –º–µ—Ç–æ–¥–æ–≤
- **–ù–µ–∏–∑–º–µ–Ω—è—é—â–∏–µ**: –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ (–æ—Ä–∏–≥–∏–Ω–∞–ª –Ω–µ —Ç—Ä–æ–≥–∞—é—Ç)
- **–ò–∑–º–µ–Ω—è—é—â–∏–µ**: –∏–∑–º–µ–Ω—è—é—Ç –∏—Å—Ö–æ–¥–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ ‚Äú–Ω–∞ –º–µ—Å—Ç–µ‚Äù

### 8.2 –¢–∞–±–ª–∏—Ü–∞ –æ–ø–µ—Ä–∞—Ü–∏–π
| –û–ø–µ—Ä–∞—Ü–∏—è | –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–µ–∏–∑–º–µ–Ω—è—é—â–∏–π | –ò–∑–º–µ–Ω—è—é—â–∏–π |
|---|---:|---|---|
| –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ | `|` | `union()` | `update()` |
| –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ | `&` | `intersection()` | `intersection_update()` |
| –†–∞–∑–Ω–æ—Å—Ç—å | `-` | `difference()` | `difference_update()` |
| –°–∏–º–º. —Ä–∞–∑–Ω–æ—Å—Ç—å | `^` | `symmetric_difference()` | `symmetric_difference_update()` |

### 8.3 –ü—Ä–∏–º–µ—Ä—ã
#### –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ
```py

```

#### –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ
```py

print(set1 & set2)

set1.intersection_update(set2)

```

#### –†–∞–∑–Ω–æ—Å—Ç—å
```py

set2 = {20, 30, 40}

print(set1 - set2)              # {10}

set1.difference_update(set2)

```

#### –°–∏–º–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å

## 9) –û—Ç–Ω–æ—à–µ–Ω–∏—è –º–µ–∂–¥—É –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏
