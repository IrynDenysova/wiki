## 6) –ú–µ—Ç–æ–¥—ã –¥–æ–±–∞–≤–ª–µ–Ω–∏—è/—É–¥–∞–ª–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤

## 8) –û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ (union / intersection / difference / symmetric difference)

### 8.1 –í–∏–¥—ã –º–µ—Ç–æ–¥–æ–≤
- **–ù–µ–∏–∑–º–µ–Ω—è—é—â–∏–µ**: –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ (–æ—Ä–∏–≥–∏–Ω–∞–ª –Ω–µ —Ç—Ä–æ–≥–∞—é—Ç)
- **–ò–∑–º–µ–Ω—è—é—â–∏–µ**: –∏–∑–º–µ–Ω—è—é—Ç –∏—Å—Ö–æ–¥–Ω–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ ‚Äú–Ω–∞ –º–µ—Å—Ç–µ‚Äù

### 8.2 –¢–∞–±–ª–∏—Ü–∞ –æ–ø–µ—Ä–∞—Ü–∏–π
| –û–ø–µ—Ä–∞—Ü–∏—è | –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–µ–∏–∑–º–µ–Ω—è—é—â–∏–π | –ò–∑–º–µ–Ω—è—é—â–∏–π |
|---|---:|---|---|
| –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ | `|` | `union()` | `update()` |
| –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ | `&` | `intersection()` | `intersection_update()` |
| –†–∞–∑–Ω–æ—Å—Ç—å | `-` | `difference()` | `difference_update()` |

### 8.3 –ü—Ä–∏–º–µ—Ä—ã
#### –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ

#### –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ

#### –†–∞–∑–Ω–æ—Å—Ç—å

#### –°–∏–º–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å

```python
        """–î–æ–±–∞–≤–ª—è–µ—Ç –¥—Ä—É–∂–±—É (–¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω—é—é)"""

        if friend not in self.friendships:
            self.friendships[friend] = set()

        """–ù–∞—Ö–æ–¥–∏—Ç –æ–±—â–∏—Ö –¥—Ä—É–∑–µ–π"""

        friends2 = self.friendships.get(person2, set())

    def suggest_friends(self, person):
        """–ü—Ä–µ–¥–ª–∞–≥–∞–µ—Ç –¥—Ä—É–∑–µ–π (–¥—Ä—É–∑—å—è –¥—Ä—É–∑–µ–π, –∏—Å–∫–ª—é—á–∞—è —É–∂–µ –¥—Ä—É–∑–µ–π)"""
```

```python
    def get_users_with_permission(self, permission):
        """–ù–∞—Ö–æ–¥–∏—Ç –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π —Å –¥–∞–Ω–Ω—ã–º —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–µ–º"""
        return {user for user, perms in self.user_permissions.items() 
                if permission in perms}

# –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ
ac = AccessControl()
ac.assign_role("–ê–ª–∏—Å–∞", "admin")
ac.assign_role("–ë–æ–±", "editor")
ac.assign_role("–í–∏–∫—Ç–æ—Ä", "viewer")

print(ac.has_permission("–ë–æ–±", "write"))  # True
print(ac.has_permission("–í–∏–∫—Ç–æ—Ä", "delete"))  # False
print(ac.get_users_with_permission("read"))  # {'–ê–ª–∏—Å–∞', '–ë–æ–±', '–í–∏–∫—Ç–æ—Ä'}
```
