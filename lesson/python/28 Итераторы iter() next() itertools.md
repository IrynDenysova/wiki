## 1) –¢–µ—Ä–º–∏–Ω—ã: iterable vs iterator
### –ò—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç (iterable)
–û–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–Ω–æ –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å –ø–æ —ç–ª–µ–º–µ–Ω—Ç–∞–º (—Å–ø–∏—Å–æ–∫, —Å—Ç—Ä–æ–∫–∞, —Å–ª–æ–≤–∞—Ä—å, set, tuple –∏ —Ç.–¥.).

### –ò—Ç–µ—Ä–∞—Ç–æ—Ä (iterator)
–û–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π:
- –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –ø–æ –æ–¥–Ω–æ–º—É,
- **–ø–æ–º–Ω–∏—Ç —Ç–µ–∫—É—â—É—é –ø–æ–∑–∏—Ü–∏—é**,
- –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º `StopIteration`.

–ò—Ç–µ—Ä–∞—Ç–æ—Ä —Å–æ–¥–µ—Ä–∂–∏—Ç:

- —Ç–µ–∫—É—â—É—é –ø–æ–∑–∏—Ü–∏—é,

## 2) –ú–∞–≥–∏—á–µ—Å–∫–∏–µ –º–µ—Ç–æ–¥—ã –∏—Ç–µ—Ä–∞—Ü–∏–∏ –∏ –∞–Ω–∞–ª–æ–≥–∏
–ò—Ç–µ—Ä–∞—Ü–∏—è –≤ Python –æ—Å–Ω–æ–≤–∞–Ω–∞ –Ω–∞ dunder-–º–µ—Ç–æ–¥–∞—Ö:

# –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∞—è —Ä–∞–±–æ—Ç–∞: –≥–µ–Ω–µ—Ä–∞—Ü–∏—è –±–µ–∑–æ–ø–∞—Å–Ω—ã—Ö –ø–∞—Ä–æ–ª–µ–π (len=4)
**–£—Å–ª–æ–≤–∏—è:**
- –¥–ª–∏–Ω–∞ –ø–∞—Ä–æ–ª—è = 4
- –º–∏–Ω–∏–º—É–º: 1 —Å—Ç—Ä–æ—á–Ω–∞—è, 1 –∑–∞–≥–ª–∞–≤–Ω–∞—è, 1 —Ü–∏—Ñ—Ä–∞
- —Å–∏–º–≤–æ–ª—ã **–Ω–µ –ø–æ–≤—Ç–æ—Ä—è—é—Ç—Å—è**
- **—Å–æ—Å–µ–¥–Ω–∏–µ —Å–∏–º–≤–æ–ª—ã –Ω–µ –º–æ–≥—É—Ç –∏–¥—Ç–∏ –ø–æ–¥—Ä—è–¥ –≤ —Ç–∞–±–ª–∏—Ü–µ —Å–∏–º–≤–æ–ª–æ–≤** (—Ä–∞–∑–Ω–∏—Ü–∞ `ord(...)` –Ω–µ —Ä–∞–≤–Ω–∞ 1)
- –≤—Å–µ –≤–∞–ª–∏–¥–Ω—ã–µ –ø–∞—Ä–æ–ª–∏ –∑–∞–ø–∏—Å–∞—Ç—å –≤ `valid_passwords.txt`

–†–µ—à–µ–Ω–∏–µ (–∏–∑ —É—Ä–æ–∫–∞, –∞–∫–∫—É—Ä–∞—Ç–Ω–æ –æ—Ñ–æ—Ä–º–ª–µ–Ω–æ):

    has_lower = any(c in ascii_lowercase for c in password)
    has_upper = any(c in ascii_uppercase for c in password)
    has_digit = any(c in digits for c in password)
    if not (has_lower and has_upper and has_digit):
        return False

        if abs(ord(password[i]) - ord(password[i + 1])) == 1:
            return False

    return True

all_variants = permutations(all_chars, 4)
valid_passwords = ("".join(p) for p in all_variants if is_valid(p))

with open("valid_passwords.txt", "w", encoding="utf-8") as f:
    f.write("\n".join(valid_passwords))

```py
import itertools

weekly_schedule = {
    "Monday": ["Gym", "Work", "Read book"],
    "Tuesday": ["Meeting", "Work", "Study Python"],
    "Wednesday": ["Shopping", "Work", "Watch movie"],
    "Thursday": ["Work", "Call parents", "Play guitar"],
    "Friday": ["Work", "Dinner with friends"],
    "Saturday": ["Hiking", "Rest"],
    "Sunday": ["Family time", "Rest"],
}

days = list(weekly_schedule.keys())
cycler = itertools.cycle(days)

while True:
    cmd = input("–ù–∞–∂–º–∏—Ç–µ 'Enter' –¥–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –ø–ª–∞–Ω–∞ (q - –≤—ã—Ö–æ–¥): ")

        break

    print(f"{day}: {plan}")
```

## –î–ó 2) –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ —Å–ø–∏—Å–∫–æ–≤ –ø—Ä–æ–¥—É–∫—Ç–æ–≤ ‚Üí –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä (–Ω–∏–∂–Ω–∏–π —Ä–µ–≥–∏—Å—Ç—Ä)
```py
from typing import Iterable, Generator

def merge_products(*lists: Iterable[str]) -> Generator[str, None, None]:
    return (item.lower() for seq in lists for item in seq)

fruits = ["Apple", "Banana", "Orange"]
vegetables = ["Carrot", "Tomato", "Cucumber"]
dairy = ["Milk", "Cheese", "Yogurt"]

gen = merge_products(fruits, vegetables, dairy)
for item in gen:
    print(item)
```
