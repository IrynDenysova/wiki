## 2) –ú–æ–¥—É–ª—å `re`: –æ—Å–Ω–æ–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏
```py

#### 1. –ë–∞–∑–æ–≤—ã–µ –ø–∞—Ç—Ç–µ—Ä–Ω—ã –∏ —Ñ–ª–∞–≥–∏
```python
import re

text = "Email: test@example.com"

pattern = r"[\w.-]+@[\w.-]+"  # \w = [A-Za-z0-9_]
match = re.search(pattern, text)
print(match.group())  # test@example.com

# –§–ª–∞–≥–∏
re.I  # ignorecase
re.M  # multiline (^ –∏ $ –Ω–∞ –∫–∞–∂–¥—É—é —Å—Ç—Ä–æ–∫—É)
re.S  # DOTALL (.) –º–∞—Ç—á –∏ \n
text = "Hello\nWorld"
print(re.findall(r"hello.world", text, flags=re.I | re.S))  # ['Hello\nWorld']
```

#### 2. –ì—Ä—É–ø–ø—ã, –∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã–µ –≥—Ä—É–ø–ø—ã, –≤—ã–±–æ—Ä–∫–∏
```python

match = re.match(r"(\d{4})-(\d{2})-(\d{2})", text)
print(match.groups())   # ('2024', '12', '31')

match = re.match(r"(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})", text)
print(match.group('year'))  # 2024

# | (–∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–∞)
print(re.findall(r"cat|dog", "cat and dog"))  # ['cat', 'dog']
```

#### 3. –ñ–∞–¥–Ω—ã–µ –∏ –ª–µ–Ω–∏–≤—ã–µ –∫–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã

#### 4. –ü—Ä–µ–¥–ø—Ä–æ—Å–º–æ—Ç—Ä—ã (lookahead/lookbehind)

## –ó–∞–∫—Ä–µ–ø–ª–µ–Ω–∏–µ 1
–°–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ:
- `\d+` ‚Üí –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ü–∏—Ñ—Ä ‚Üí **c**
- `[a-zA-Z0-9]+` ‚Üí –±—É–∫–≤—ã –∏ —Ü–∏—Ñ—Ä—ã ‚Üí **b**
- `\s+` ‚Üí –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø—Ä–æ–±–µ–ª—å–Ω—ã—Ö ‚Üí **a**
- `[^a-zA-Z0-9]+` ‚Üí –ù–ï –±—É–∫–≤—ã –∏ –ù–ï —Ü–∏—Ñ—Ä—ã ‚Üí **d**

–û—à–∏–±–∫–∞ –≤ —à–∞–±–ª–æ–Ω–µ:

```
–ü—Ä–∞–≤–∏–ª—å–Ω–æ: –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å raw string ‚Üí `re.findall(r"\d+", ...)`

–õ–µ–Ω–∏–≤—ã–π –∫–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä: **`<.*?>`**

## –ó–∞–∫—Ä–µ–ø–ª–µ–Ω–∏–µ 2
–Ø–∫–æ—Ä—è:
- `^` ‚Üí –Ω–∞—á–∞–ª–æ —Å—Ç—Ä–æ–∫–∏
- `$` ‚Üí –∫–æ–Ω–µ—Ü —Å—Ç—Ä–æ–∫–∏
- `\b` ‚Üí –≥—Ä–∞–Ω–∏—Ü–∞ —Å–ª–æ–≤–∞
- `\B` ‚Üí –Ω–µ –≥—Ä–∞–Ω–∏—Ü–∞ —Å–ª–æ–≤–∞

–û—Ç–ª–∏—á–∏–µ `re.match()` –æ—Ç `re.search()`:
- `match()` –ø—Ä–æ–≤–µ—Ä—è–µ—Ç **—Ç–æ–ª—å–∫–æ –Ω–∞—á–∞–ª–æ**,
- `search()` –∏—â–µ—Ç **–≤ –ª—é–±–æ–π —á–∞—Å—Ç–∏** —Å—Ç—Ä–æ–∫–∏.
