### 2.1 –ß—Ç–æ —Å—á–∏—Ç–∞–µ—Ç—Å—è True (truthy)
- –ª—é–±–æ–µ **–Ω–µ–Ω—É–ª–µ–≤–æ–µ** —á–∏—Å–ª–æ (`42`, `-2`, `3.14`)
- –ª—é–±–∞—è **–Ω–µ–ø—É—Å—Ç–∞—è** —Å—Ç—Ä–æ–∫–∞ (`"0"`, `"Python"`, `"False"`)

### 2.2 –ß—Ç–æ —Å—á–∏—Ç–∞–µ—Ç—Å—è False (falsey)
- `None`
- `0`
- –ø—É—Å—Ç—ã–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏/–∫–æ–ª–ª–µ–∫—Ü–∏–∏: `""`, `[]`, `{}` –∏ —Ç.–ø.

## 3) –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è
–û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –≤ Python –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç **–≤—Å–µ–≥–¥–∞** `True` –∏–ª–∏ `False`.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –°–º—ã—Å–ª |

| `<=` | –º–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ |

–ü—Ä–∏–º–µ—Ä—ã:

## 12) –î–æ–º–∞—à–Ω–µ–µ –∑–∞–¥–∞–Ω–∏–µ ‚Äî –ø–æ–¥—Å–∫–∞–∑–∫–∏ —Ä–µ—à–µ–Ω–∏–π

### 12.1 –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ + —Å—Ä–∞–≤–Ω–µ–Ω–∏—è (and/or/not + ==/!=)

### 12.2 –ü—Ä–æ–≤–µ—Ä–∫–∞ —É—Å–ª–æ–≤–∏–π (—Å–≤–µ—Ç / –æ–∫–Ω–æ)
–ó–∞–¥–∞—á–∞: –¥–≤–∞ –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è ‚Üí –ø—Ä–æ–≤–µ—Ä–∏—Ç—å:
- –æ–±–∞ –ª–∏ —É—Å–ª–æ–≤–∏—è –≤—ã–ø–æ–ª–Ω–µ–Ω—ã
- —Ö–æ—Ç—è –±—ã –æ–¥–Ω–æ –≤—ã–ø–æ–ª–Ω–µ–Ω–æ 

```py
light_on = input("–°–≤–µ—Ç –≤–∫–ª—é—á—ë–Ω? True/False: ").strip().lower() == "true"
```

### 12.3 (*) –°—Ç–æ–∏–º–æ—Å—Ç—å –º–æ–±–∏–ª—å–Ω–æ–≥–æ —Ç–∞—Ä–∏—Ñ–∞
–£—Å–ª–æ–≤–∏—è: –±–∞–∑–æ–≤–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å 30‚Ç¨, –≤—Å—ë —Å–≤–µ—Ä—Ö 500 –ú–ë ‚Äî –ø–æ 0.2‚Ç¨ –∑–∞ –ú–ë.

## –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–∞—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è

### –í–∞–∂–Ω—ã–µ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ –¥–ª—è –∏–∑—É—á–µ–Ω–∏—è

#### 1. –ö–æ—Ä–æ—Ç–∫–æ–µ –∑–∞–º—ã–∫–∞–Ω–∏–µ (Short-circuit evaluation)
Python –æ—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç –ø—Ä–æ–≤–µ—Ä–∫—É, –∫–∞–∫ —Ç–æ–ª—å–∫–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∏–∑–≤–µ—Å—Ç–µ–Ω:
```python
# and: –µ—Å–ª–∏ –ø–µ—Ä–≤–æ–µ False, –≤—Ç–æ—Ä–æ–µ –Ω–µ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è
def check():
    print("–ü—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è!")
    return True

result = False and check()  # "–ü—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è!" –ù–ï –≤—ã–≤–æ–¥–∏—Ç—Å—è
print(result)  # False

# or: –µ—Å–ª–∏ –ø–µ—Ä–≤–æ–µ True, –≤—Ç–æ—Ä–æ–µ –Ω–µ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è
result = True or check()    # "–ü—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è!" –ù–ï –≤—ã–≤–æ–¥–∏—Ç—Å—è
print(result)  # True
```

#### 2. –í–æ–∑–≤—Ä–∞—Ç –∑–Ω–∞—á–µ–Ω–∏–π –∏–∑ –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π
`and` –∏ `or` –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ `True`/`False`:
```python
# or –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–µ—Ä–≤–æ–µ True-–∑–Ω–∞—á–µ–Ω–∏–µ –∏–ª–∏ –ø–æ—Å–ª–µ–¥–Ω–µ–µ
print("text" or "default")    # "text"
print("" or "default")        # "default"
print(0 or 5 or 10)          # 5

# and –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–µ—Ä–≤–æ–µ False –∏–ª–∏ –ø–æ—Å–ª–µ–¥–Ω–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ
print("a" and "b" and "c")   # "c"
print("a" and 0 and "c")     # 0

# –ü—Ä–∏–º–µ–Ω–µ–Ω–∏–µ: –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é
name = input("–ò–º—è: ") or "–ê–Ω–æ–Ω–∏–º"
```

#### 3. –°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤
```python
# –ß–∏—Å–ª–∞ –º–æ–∂–Ω–æ —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å –º–µ–∂–¥—É —Ç–∏–ø–∞–º–∏
print(5 == 5.0)    # True
print(5 < 5.1)     # True

# –°—Ç—Ä–æ–∫–∏ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ª–µ–∫—Å–∏–∫–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏ (–ø–æ Unicode)
print("apple" < "banana")  # True
print("Apple" < "apple")   # True (–∑–∞–≥–ª–∞–≤–Ω—ã–µ –∏–¥—É—Ç —Ä–∞–Ω—å—à–µ)
print("10" < "2")          # True (—Å—Ç—Ä–æ–∫–∏, –Ω–µ —á–∏—Å–ª–∞!)

# None –º–æ–∂–Ω–æ —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å —Ç–æ–ª—å–∫–æ —Å None
print(None == None)  # True
print(None == 0)     # False
```

#### 4. –û–ø–µ—Ä–∞—Ç–æ—Ä `is` vs `==`
```python
# == –ø—Ä–æ–≤–µ—Ä—è–µ—Ç —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –∑–Ω–∞—á–µ–Ω–∏–π
# is –ø—Ä–æ–≤–µ—Ä—è–µ—Ç –∏–¥–µ–Ω—Ç–∏—á–Ω–æ—Å—Ç—å –æ–±—ä–µ–∫—Ç–æ–≤ (–æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –æ–±—ä–µ–∫—Ç –≤ –ø–∞–º—è—Ç–∏)

a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a is b)  # False (—Ä–∞–∑–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã)
print(a is c)  # True (—Å—Å—ã–ª–∞—é—Ç—Å—è –Ω–∞ –æ–¥–∏–Ω –æ–±—ä–µ–∫—Ç)

# –î–ª—è None –≤—Å–µ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ is
x = None
if x is None:  # ‚úÖ –ü—Ä–∞–≤–∏–ª—å–Ω–æ
    print("x —Ä–∞–≤–µ–Ω None")

    print("x —Ä–∞–≤–µ–Ω None")
```
