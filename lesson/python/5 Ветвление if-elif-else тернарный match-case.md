## 11) `match-case` (Python 3.10+)
`match-case` —É–ø—Ä–æ—â–∞–µ—Ç ‚Äú–º–Ω–æ–≥–æ –≤–∞—Ä–∏–∞–Ω—Ç–æ–≤‚Äù –≤–º–µ—Å—Ç–æ –¥–ª–∏–Ω–Ω–æ–≥–æ `if-elif-elif...`.

```py

    case _:
```
