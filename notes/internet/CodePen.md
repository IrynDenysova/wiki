Вот самые близкие **аналоги CodePen** (выбирай по задаче):

### Как CodePen: быстро HTML/CSS/JS + шаринг

- **JSFiddle** — классический “fiddle”: быстро набросать пример и поделиться ссылкой. ([jsfiddle.net](https://jsfiddle.net/?utm_source=chatgpt.com "JSFiddle - Code Playground"))
    
- **JS Bin** — тоже лёгкий playground, удобно для дебага/демо и встраивания (embed). ([JS Bin](https://jsbin.com/?utm_source=chatgpt.com "JS Bin - Collaborative JavaScript Debugging"))
    
- **PlayCode** — JS-песочница с live preview, multi-file и npm-пакетами. ([PlayCode.io](https://playcode.io/?utm_source=chatgpt.com "JavaScript Playground - Free Online JS Sandbox"))
    

### “Проектно”: npm, React/Vue/Angular, файловая структура, как мини-IDE

- **StackBlitz** — браузерная IDE с “инстант” окружениями; основной упор на современный фронт и быстрый старт. ([StackBlitz](https://stackblitz.com/?utm_source=chatgpt.com "StackBlitz | Instant Dev Environments | Click. Code. Done."))
    
- **CodeSandbox** — облачные dev-окружения + удобный шаринг и коллаборация по ссылке. ([CodeSandbox](https://codesandbox.io/?utm_source=chatgpt.com "CodeSandbox: Instant Cloud Development Environments"))
    

### Если нужен не только фронт, но и разные языки/деплой

- **Replit** — браузерная IDE, совместная работа и публикация/деплой проектов. ([Replit Docs](https://docs.replit.com/replit-app/collaborate?utm_source=chatgpt.com "Collaborate in real time (Join Links)"))
    

### Узкоспециализированно (иногда очень удобно)

- **TypeScript Playground** — быстро проверить/поделиться TypeScript-кодом. ([TypeScript](https://www.typescriptlang.org/play/?utm_source=chatgpt.com "TS Playground - An online editor for exploring ..."))
